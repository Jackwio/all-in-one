# All-in-One 功能總覽

## 專案作用
此專案是一個靜態前端展示頁，透過「分類頁籤 + 功能卡片」集中呈現 All-in-One 工具的能力地圖。
主要目的在於讓使用者能在單一頁面快速理解各模組定位、狀態與入口連結，降低工具探索與切換成本。

## 主要內容
- 分類瀏覽：核心工具、開發能力、洞察分析、筆記
- 卡片資訊：功能名稱、狀態、說明、標籤與導向連結
- 預設焦點：載入時預設切到「筆記」分類，方便快速查閱知識內容

## 專案結構
- `index.html`：頁面骨架與區塊結構
- `assets/app.js`：資料定義與 Tabs/Cards 渲染邏輯
- `assets/styles.css`：視覺樣式與響應式版面
- `tests/page-structure.test.sh`：頁面結構與關鍵字串檢查

## 本機查看與測試
1. 以瀏覽器直接開啟 `index.html`。
2. 執行測試命令：

```bash
bash tests/page-structure.test.sh
```
